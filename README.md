# VIBE-ATHON 

An AI-powered Classical Music Analyzer project developed for hackathon  I participate VIVE-ATHON hackathon and learning purposes.

##  Project Overview
VIBE-ATHON is a smart music analysis system that analyzes classical music and provides insights such as mood, genre, and audio-based predictions using AI/ML concepts.
## Features
- Classical music analysis
- Emotion/Mood detection
- Audio processing
- AI/ML prediction integration
- Interactive frontend UI
- Real-time result display



##  Technologies Used

### Frontend

- CSS
- JavaScript

### Backend

- Flask

### AI/ML
 Machine Learning
 Audio Feature Extraction
